# 1\) Логика рекламного кабинета

Для этой ниши я бы не делал одну большую кампанию.  
Правильнее — разделить по интенту и типу спроса.

## Рекомендуемая верхнеуровневая структура

1. Поиск — Общие коммерческие  
2. Поиск — Размеры  
3. Поиск — Дизайны / фактуры  
4. Поиск — По применению  
5. Поиск — Свойства / поверхности  
6. Поиск — Гео  
7. Поиск — Бренды производителей  
8. Поиск — B2B / опт / объекты  
9. РСЯ — Ретаргетинг  
10. РСЯ — Интерес / широкий охват  
11. Мастер кампаний / смарт-эксперименты — только после накопления данных

# 4\) KPI, UTM и правила оптимизации

## Основные KPI

Для этой ниши я бы смотрел не только лиды, но и промежуточные сигналы.

### Макроконверсии:

* звонок  
* заявка  
* заказ  
* запрос цены  
* запрос на подбор  
* обращение в WhatsApp / Telegram  
* отправка формы “получить расчет”

### Микроконверсии:

* просмотр 3+ страниц  
* просмотр страницы категории 60x60 / 60x120 / под мрамор  
* просмотр карточки товара  
* добавление в корзину  
* клик по номеру  
* клик по мессенджеру  
* скачивание прайса  
* отправка формы “нужна консультация”

---

## Целевые ориентиры по метрикам

Без вашей аналитики это только ориентир, но как старт норм:

### Для поиска:

* CTR: 8–15%+ на горячих группах  
* Доля отказов: чем ниже, тем лучше  
* Конверсия в лид: условно 1.5–5%+, зависит от сайта  
* Конверсия в микроконверсию: 8–20%+

### Для РСЯ:

* CTR обычно ниже  
* главный KPI — цена микроконверсии / ретаргет-лида

---

## Нейминг кампаний

Чтобы кабинет не развалился через месяц, сразу задайте стандарт.

### Формат:

СЕТЬ | ТИП | НАПРАВЛЕНИЕ | ГЕО

Примеры:

* ПОИСК | Общие | Купить\_Цена\_Каталог | Москва  
* ПОИСК | Размеры | 60x60 | Москва  
* ПОИСК | Дизайн | Под\_Мрамор | Москва  
* РСЯ | Ретаргетинг | Все\_посетители | РФ

---

## Правила оптимизации после запуска

### Через 7–10 дней:

смотрите:

* какие группы дают клики без микроконверсий  
* какие поисковые фразы мусорные  
* какие страницы плохо удерживают трафик

### Что делать:

1. Добавлять минус-слова  
2. Отключать слабые фразы  
3. Повышать ставки на группы с заявками  
4. Выносить лучшие фразы в отдельные группы  
5. Делать отдельные объявления под сильные связки  
   * 60x60  
   * 60x120  
   * под мрамор  
   * для пола  
   * матовый  
   * Москва

---

## Что запускать первым делом

Если нужен минимально жизнеспособный кабинет, стартуйте с 8 кампаний:

1. Общие коммерческие  
2. 60x60  
3. 60x120  
4. Под мрамор  
5. Под бетон  
6. Для пола  
7. Для ванной  
8. Москва / ваш главный регион

Это даст самую быструю обратную связь.
